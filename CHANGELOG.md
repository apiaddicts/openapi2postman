# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).


## [2.4.1-beta-1] - 2026-06-26

### Fixed
- Fix crash when `securitySchemes` contains `type: apiKey` or other non-OAuth2 schemes.


## [2.4.0] - 2026-05-07

### Added
- Added support for OpenAPI 3.2.

## [2.3.0] - 2026-03-05

### Added
- Added support for OpenAPI 3.1.

## [2.1.0] - 2024-10-08

## Added

### Microcks headers added

- Added 'microcks_headers' parameter in the environments config section to auto configure the Micrkocs headers.
